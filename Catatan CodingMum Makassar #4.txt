/* catatan, komentar, tidak akan memberi efek */

color:
- bisa pakai nama warna (bahasa Inggris), hanya ada 64 warna yang didefinisikan namanya
- bisa pakai rgb( R, G, B) dimana R, G, B adalah skala masing-masing warna (0-255)
- bisa pakai heksadesimal #RRGGBB dimana RR, GG, BB adalah skala masing-masing warna dalam heksadesimal (00-FF)

format style dalam stylesheet: selector { properties: value; }
(lihat buku modul)

.namaselector --> diawali dengan titik, dalam HTML dipanggil sebagai class (class="...") 
#namaselector --> diawali dengan hash (#), dalam HTML dipanggil sebagai id (id="...")
namaselector  --> tanpa tanda = berlaku untuk setiap tag yang namanya sama dengan namaselector 

class bisa dipanggil berulangkali pada sejumlah tag HTML dalam satu file HTML yang sama
id hanya milik satu tag dalam sebuah file HTML

mendefinisikan style langsung untuk sebuah tag tidak dianjurkan untuk tag yang sering/banyak kali digunakan dan kemungkinan membutuhkan style berbeda, gunakan class.
<body> bisa langsung didefinisikan style-nya karena hanya ada satu dalam satu file HTML
